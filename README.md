## 简介 
   <h2 align="center">Fastgo 能够在瞬间创建一个MVP+Dagger2+Retrofit+RxJava的Android Project。</h2>
   
   
## 使用 
```
Gradle:
compile 'me.walten:fastgo:1.0.7'

Maven:
<dependency>
  <groupId>me.walten</groupId>
  <artifactId>fastgo</artifactId>
  <version>1.0.7</version>
  <type>pom</type>
</dependency>

```
